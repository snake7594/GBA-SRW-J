=====================================================================
 슈퍼로봇대전 J (GBA) — 전투 대사 한글 패치 도구 (완성판)
 단일 편집 마스터 + 합체기(blk193) 길이보존 통합
=====================================================================

'한글 폰트 + 시나리오 + 타이틀'이 적용된 32MB ROM에 전투 대사를
한글로 입히는 PC용 도구입니다. 합체기(합동공격) 대사까지 포함합니다.

★ 편집할 파일은 battle_dialogue_unique.json 하나입니다.
★ 합체기(blk193)는 패치할 때 자동으로 '길이보존' 처리되어 '도몬 버그'가
  나지 않습니다. 별도 작업이 필요 없습니다.

---------------------------------------------------------------------
[1] 워크플로 (4단계)
---------------------------------------------------------------------
① 번역 수정 : battle_dialogue_unique.json 의 "tr" 만 편집
     (jp=마커 뗀 원문, 수정 금지 / tr=한국어 번역)
② (선택) 줄바꿈 정리 : python rewrap_battle.py battle_dialogue_unique.json tr
③ 변환 : python build_battle_json.py input.gba battle_dialogue.json battle_dialogue_unique.json
④ 패치 : python 패치하기.py
     - 일반 대사 → 확장영역 재배치
     - 합체기(blk193) → 원위치 동일 길이로 패치(길이보존)

---------------------------------------------------------------------
[2] 합체기(blk193) 길이보존 — 왜 필요한가
---------------------------------------------------------------------
blk193은 게임 내 유일한 합동공격 전용 블록(천공검·그란대셔·갓핑거 등)으로,
연출(화자 얼굴 전환)이 각 대사의 '바이트 길이'에 의존합니다. 길이가
바뀌면 화자가 '도몬'으로 고정되고 대사가 깨집니다.

패치 엔진(srwj_battle_kr_insert.py)은 blk193만 다음과 같이 처리합니다:
  - 재배치하지 않고 '원위치'에 패치 (인덱스 불변)
  - 번역이 원본보다 짧으면 → 공백을 채워 길이를 정확히 맞춤
  - 번역이 원본보다 길면 → 그 토큰만 원문(일본어) 유지
  ※ LEN_PRESERVE_BLOCKS = {193} 상수로 제어

★ 중요: 합체기 대사는 원본 일본어보다 '길면 안 됩니다'.
  battle_dialogue_unique.json 의 합체기 대사는 이미 원본 길이에 맞춰
  축약돼 있습니다(예: 단숨에 간다→확간다, 거머쥐라고→잡아). 추가 수정
  시 원본 길이를 넘지 마세요. 참고: blk193_축약사전.json

[3] 파일 목록
---------------------------------------------------------------------
[편집 / 변환]
  battle_dialogue_unique.json   편집 마스터 (jp↔tr, 합체기 축약 포함)
  build_battle_json.py            unique + ROM → battle_dialogue.json
  rewrap_battle.py                줄바꿈 정리(14칸 규칙)
  battle_dialogue.json            build 결과(패치 입력본)
  blk193_축약사전.json            합체기 축약 참고 (jp → 축약 ko)
[패치 코어]
  패치하기.py                     battle_dialogue.json → ROM 패치
  srwj_battle_kr_insert.py        삽입 엔진 (blk193 길이보존 통합)
  srwj_battle_codec.py            전투 대사 코덱
  srwj_archive.py                 블록 아카이브
  verify_battle.py                검증
[무기명 — 별도 시스템(참고용)]
  srwj_weapon_patch.py, _wpn_tr.py, _wrap.py, _weapon_names.json,
  _untrans_real.json, _not_in_rom.json, kouji_tr.py, changes_report.txt

---------------------------------------------------------------------
[4] 주의사항
---------------------------------------------------------------------
- jp(원문) 수정 금지 (매칭 키)
- 합체기(blk193) 대사는 원본 길이 초과 금지 (길이보존 제약)
- tr 에 카타카나·히라가나 남으면 미완성 번역
- ROM 은 32MB 한글 통합본 사용
