
<제 11화 - 사막의 호랑이(砂漠の虎)>



[인터미션 - 육상전함 레셉스/브릿지]
다코스타 : 발트펠드 대장님.

발트펠드 : 무슨 일인데 그렇게 서두르나, 다코스타군. 목성 도마뱀이나 에일리언이 공
               동전선이라도 펼친 건가? 그럼 놀랄만도 하다만.

다코스타 : 오늘 아침경에 이동을 개시한 예의 2척 말입니다만, 현재 각각 다른 방면으로
               이동중이란 보고가 들어왔습니다.

발트펠드 : 둘로 나뉘었어? 이 상황 하에서? 과연. 그래서, 어느 쪽이 더 확 들어오지?

다코스타 : 예?

발트펠드 : 어느 쪽이 더 눈에 띄게 행동하고 있냐고.

다코스타 : 눈에 띄게, 말인가요. 나데시코 쪽은 루트상에 있던 튤립을 하나 박살냈습니
               다. 예의 그래비티 블라스트의 사용도 확인되었습니다. 현재 우군이 추격중
               입니다.

발트펠드 : 과연.

다코스타 : 그래서, 어쩌실 겁니까?

발트펠드 : 목표는 아크엔젤이다. 아무래도 나데시코는 아크엔젤에게 적을 보내고 싶지
               않은 것 같으니까.

다코스타 : 예, 알겠습니다.

아이샤(アイシャ) : 즐거워 보이네, 앤디.

발트펠드 : 요즘은 레지스탕스나 인간이 아닌 것들만 상대했었으니까. 정상적인 상대는
               오랜만이야. 소문의 일행 중 일부, 그 실력을 구경해보기로 하지. 자, 전쟁을
               하러 가자.



-아크엔젤/브릿지-
마류 : 일단 아직까지는 순조로운 것 같네.

나탈 : 예. 하지만, 우리가 인도양으로 빠지기 위해서는 자프트군이 있는 지대를 지나가
         지 않으면 안됩니다. 어느 정도의 적이 모여있는지, 그다지 낙관시 할 수는 없겠
         죠.

마류 : 알고있어. 그걸 위해 그들이 동행해준 거잖아.

나탈 : ......

무우 : 그 카가리라는 아가씨는 왜 이쪽에 탄 거야? 수전기대들은 나데시코로 가버렸는
         데.

마류 : 글쎄. 도울 수 있는 일 정도는 있을 거라며 오긴했지만. 이제 와서 그녀만 안된다
         고 할 수도 없었어.

무우 : 뭐, 그건 그렇다만.

토르 : 수고하셨습니다. 프라가 대위님, 교대예요.

무우 : 아, 고마워. 그럼 난 잠깐 그들이라도 보고 올까.

마류 : 예, 부탁해요.



-아크엔젤/거주구/병사 숙소-
훼스테니아 : 아~ 이럴줄 알았다면 나데시코에 남을걸. 토우야 때문이야. 밥도 적고.

토우야 : 내게 맡긴다고 한건 너잖아.

훼스테니아 : 이럴 줄 몰랐으니까지!

메르아 : 테니아...

카티아 : 테니아, 적당히 해. 대체 그 소리만 몇 번째니.

미리아리아 : 그런데, 나데시코는 그렇게 달라?

훼스테니아 : 그럼! 방도 우리들 셋이 하나를 쓰고, 식사는 메뉴도 풍부하고 언제라도 먹
               을 수 있고, 욕실도 넓어. 모든게 나데시코 쪽이 훨씬 좋아.

사이 : 어쩔 수 없잖아. 아크엔젤은 군함이니까.

로안 : 그래요. 나데시코와는 달라서, 우리 같은 민간인이 타는 경우는 고려되지 않았으
         니까요.

시모느 : 하지만, 확실히 나데시코 쪽이 좋았어.

데이빗 : 뭐, 아무렴 어때. 어쨌든 여긴 이미 지구고, 조금만 더 참자구.

아서 : 아~아, 어서 이런 위험한 일과는 이별하고 집으로 돌아가고 싶어.

미리아리아 : ......

시모느 : 아서.

아서 : 아... 미안. 그러고 보니, 너희는 헤리오폴리스에...

시모느 : 넌 정말 무신경하다니까. 그러니 여자애들에게 인기가 없지.

아서 : 너무해, 시모느.

사이 : 아, 프레이. 어디 갔었어?

프레이 : 응... 잠깐.



-아크엔젤/브릿지-
노알 : 아, 왔군, 토우야.

토우야 : 얘기할 거란게 뭐죠?

무우 : 아니, 별로 대단한 건 아니다만, 잠시 함께 싸우게 될 테니까, 파일럿 제군과는 한
         번쯤 제대로 얘길 해볼까 해서.


(대화 중)


무우 : 흐음, 그랬었군. 그쪽도 꽤나 고생이었구나.

키라 : ...화성이 전멸하다니, 정말 그렇게 심각한 상황이에요?

데이빗 : 그래, 최악이었지.

무우 : 하지만, 넌 정말 괜찮은 거야? 아무리 부친이 지구인이라고 해도, 그 그라도스란
         별에서 태어나고 자랐잖아. 아까 말한 게일이란 사람말고도 아는 사람도 많을 테
         고.

에이지 : ...지구는 제 동경의 대상, 마음의 고향이었습니다. 전 이 별을 지키고 싶어요.
            어떻게든 그라도스와 지구의 싸움을 막고 싶습니다.

무우 : 지구의 사람이 다들 너 같이 생각해주면 좋겠는데. 정말 부끄럽기 짝이 없군. 그
         런데, 그 D보이라는 사람은 어때?

노알 : 녀석이 뭘 생각하는지 어떻게 압니까. 테카맨이란 걸로 변신하고, 에이지나 그라
         도스 녀석들이 지구인이라고 하는게 더 신빙성 있겠어요.

아키 : 노알, 아직도 D보이를.

노알 : 의심하고 있어. 믿으라는게 무리지. 조금이라도 수상한 짓을 하면, 난 언제라도 녀
         석을 쏠거야.

무우 : 뭐, 얘기를 들어보니, 라담과 싸우는 동안은 아군이 되어줄 것 같은데. 그러고 보
         니, 시운 토우야군.

토우야 : 예?

무우 : 아니, 이런저런 일로 얼렁뚱땅 이렇게 되버렸지만. 그 때 목성 도마뱀과 싸우고,
         우리도 공격해왔던 녀석들이 뭐였는지, 네 일행은 알고 있겠지? 그 얘기좀 들려
         줘.

토우야 : ...저도 잘은 몰라요. 그저, 카티아들이 도망쳐온 상대 같다는 정도.

키라 : 도망쳐와? 그 애들이 도망쳐왔단, 무슨 소리예요?


(대화 중)


무우 : 과연. 그런 일이 있었군. 그걸로 네 기체나 그녀들이 동승하고 있는 것도 조금은
         알겠어. 하지만, 결국 그 녀석들이 뭔지는 알아낸게 없군.



-아크엔젤/격납고-
무우 : 하아, 이걸로 일단 체크는 끝났나. 중사, 나중에 확인해줘.

마독 : 알겠습니다.

무우 : 응? 중사, 저 녀석들, 뭐하는 거야?

마독 : 아아, 스카이글래스퍼 시뮬레이터를 사용 중인가봅니다. 그 카가리라는 아가씨가
         성적이 좋아서, 다른 녀석들도 모인 것 같아요.

무우 : 그러고 보니, 스카이글래스퍼가 하나 더 있었지?

마독 : 대위님, 설마 태우겠다는 건 아니시겠죠?

무우 : 당연하지. 서툰 아군기는 되려 방해만 돼.

마독 : 그렇게 말씀하시니 안심이군요.

카가리 : 아, 제길. 당해버렸나. 뭐, 이쯤 해 둘까.

미리아리아 : 하지만, 굉장하다.

토르 : 우리들은 아주 형편없었는데. 카가리랬지? 전투기에 타본적 있어?

카가리 : 없지만, 그런 것쯤 간단하지. 너희들이 너무 한심한 것뿐이야. 그래서는 죽는다.
            전쟁 중이니까.

데이빗 : 오호, 말 한번 잘하는군. 그럼 이번엔 내가 해볼까.

시모느 : 데이빗, 그만두는게 좋을 텐데? 행여나 SPT 파일럿이 그녀보다도 못하면 꼴불
            견이야.

로안 : 시모느가 해보는 건 어때요?

카가리 : 그래, 나도 보고 싶어. 화성에서 살아남은 녀석의 실력이란 걸 보여달라구.

시모느 : 됐어. 하지만 SPT라면 또 다뤄보고 싶어.

데이빗 : 나와 로안이 있잖아. 시모느가 탈 일은 없을걸.

시모느 : 하지만, 해두면 너희들이 다쳐서 나갈 수 없을 때라던가, 도움이 되잖아?

미리아리아 : 저것도 시뮬레이터가 있어?

데이빗 : 직접 하는 거야. 실전 기록을 사용한 시뮬레이션이 가능해. 화성에선 우리도 그
            걸로 연습했지.

카가리 : 헤에, 재밌을거 같은데. 조금만 시켜줘라.

로안 : 안돼요. 에이지에게 허락 받지 않으면.

카가리 : 받음 되지. 에이지는 어딨어?

무우 : 이봐, 이봐, 너희들. 열심인 건 좋지만, 실전과 시뮬레이션을 전혀 다르다. 행여나
         난데없이 멋대로 출격해버리는 짓만은 삼가해줘.

데이빗 : 확실히, 우리가 갑작스런 실전에서도 죽지 않고 끝난 건 운이 좋았던 것과 에이
           지 덕분이었으니까.

로안 : 그렇군요. 지금 생각해보면 용케도 살아있어요, 우리들.

카가리 : 아, 알아, 그쯤은.



-아크엔젤/브릿지-
마류 : 뭐라고?

나탈 : 틀림없습니다. 자프트에게 포착되었습니다. 정찰에 모빌슈츠를 내보낸 것으로 봐,
        근처에 모함이 있을 겁니다.

마류 : 전투는 피할 수 없을 것 같네. 미안, 얘기를 듣고 싶었는데 그럴 여유가 없을것 같
         아. 당신도 싸워 주겠지, D보이군?

D보이 : 흥미 없어.

아키 : D보이.

D보이 : 라담이 침략을 개시하고 라담 나무의 숲까지 있는데, 지구인끼리 싸우다니, 참
           으로 어리석군. 아무리 달라도 라담에 비하면 코디네이터는 지구인 그 자체잖아?

마류 : 그러네. 네 말대로야. 연합과 플랜트는 목성 도마뱀이 있는데도 개전했어, 그것만
         으로도 충분히 어리석지. 하지만, 그래서 어쩌라는 거야? 지금의 우린 싸우지 않
         는다는 선택지는 없어.

무우 : 그런 상황이 되버린 이상, 싸우지 않으면 지킬 수 없는 것도 있으니, 그래서 싸우
         는 거야. 관심 없다고 끝날 일이면 나도 그러고 싶다. 다른 녀석들도 그렇겠지. 특
         히 군인이 아닌 자들은 말이야.

D보이 : ......

노알 : 소용없습니다, 대위님. 이 녀석은 라담과 싸울 수만 있다면 다른건 아무래도 좋은
         녀석이에요. 그라도스에게 침략 당하던, 지상이 어떻게 되던. 어이, D보이. 이왕
         이면 자프트에라도 가는게 어때?

아키 : 노알, 무슨 소리야!? D보이는 이미 우리 동료잖아.

D보이 : ...어쨌든 싸워는 줄거야. 이 함이 당한 뒤가 귀찮아지니까. 하지만 내게 동료는
           필요 없어.

나탈 : 그래도 상관없어. 즉시 나갈 준비를 해둬라.

아키 : 잠깐만요, 바질 소위님.

나탈 : 문답할 여유가 없어. 그가 싸우겠다면 문제는 없겠지.


(경보)


사이 : 자프트군 모빌슈츠隊의 접근을 확인!

마류 : 왔군. 전원 제1 전투배치!



-아크엔젤/격납고-
프레이 : 키라.

키라 : 프레이, 전투가 될 꺼야. 이런데 있으면 위험해.

프레이 : 응... 키라, 지켜줘. 그 녀석들, 전부 해치워버려.

키라 : 알았어, 프레이. 이제 아무도 죽게 하지 않겠어. 내가 지켜낼 테니까.





[전투 맵]
사이 : 자프트군 모빌슈츠, 진, 바쿠, 딘을 확인.

마류 : 발진을 서둘러.

미리아리아 : 전기 발진해주세요. 다들, 조심해.


(아군 출격)


키라 : 아크엔젤은 지켜내겠어. 당하게 두진 않아!

무우 : 꽤나 기세 좋은거 같은데. 하지만, 모함이 나오지 않았다는 건, 후속이 있다는 거
         겠지. 다들, 무리는 하지 마라.

토우야 : 라져. ...자프트라. 저 녀석들이 지상을 공격하지만 않았어도, 라담을 물리쳤을
            지도 모르는데.



-적 일정 수 이하-
카가리 : 놔! 왜 내가 나가면 안된다는 거야! 저기에 하나가 비잖아! 전력을 굴릴 여유가
            있는 상황이야!?

마독 : 대위님이 말했잖냐!? 죽으려고 나가 봤자 방해라고!

카가리 : 난 죽을 생각 없어.

시모느 : 조금은 진정해. 걱정마, 에이지들이 싸워줄 테니까.

카가리 : 큭...



-적 일정 수 이하-
사이 : 이건... 미사일입니다! 직격으로 옵니다!

(아크엔젤 피탄)

마류 : 윽.

나탈 : 크, 색적, 뭘하고 있나! 모함이 있다!
